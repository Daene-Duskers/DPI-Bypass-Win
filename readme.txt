Запуск и остановка службы строго от имени администратора.
При проблемах с сайтами и приложениями остановите службу!

Запуск службы Windows:    zapret-winws/service_create.cmd
Остановка службы Windows: zapret-winws/service_del.cmd

--------------------------------------------------------------

Можно разблокировать свои сайты и ресурсы добавив их в список.
Для подхвата ресурсов из списка требуется перезапуск браузера.

Добавить свои ресурсы:    zapret-winws/files/list-resource.txt
